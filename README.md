# 深圳市未莱人工智能软件开发有限公司 官网

## 项目简介

这是深圳市未莱人工智能软件开发有限公司的官方网站，展示公司的核心业务、产品和企业文化。

## 主要特性

- ✨ 现代化科技感设计
- 🎨 渐变色彩和动画效果
- 📱 完全响应式布局，适配所有设备
- 🚀 流畅的页面滚动和交互体验
- 💡 展示AI+应用和软件外包服务
- 🎯 突出心语App产品特色
- 🖼️ 支持真实 App 截图画廊展示

## 技术栈

- HTML5
- CSS3 (现代特性：Grid, Flexbox, 渐变, 动画)
- 原生JavaScript (ES6+)

## 快速开始

1. 直接在浏览器中打开 `index.html` 文件即可查看网站

2. 或者使用本地服务器运行（推荐）:
   ```bash
   # 使用Python
   python -m http.server 8000
   
   # 使用Node.js
   npx serve
   ```

3. 在浏览器中访问 `http://localhost:8000`

## 文件结构

```
xinyuchat/
├── index.html               # 主页面
├── style.css               # 样式文件
├── script.js               # 交互脚本
├── assets/
│   └── screens/            # 心语App截图（需自行放置）
└── README.md               # 项目说明
```

### 心语App截图资源

- 将提供的四张截图放入 `assets/screens/` 目录。
- 建议按以下文件名命名，以便页面自动加载：
    - `xinyu-chat.jpg`（“我说”对话入口）
    - `xinyu-profile.jpg`（“我的”中心）
    - `xinyu-insight.jpg`（情绪洞察）
    - `xinyu-growth.jpg`（心灵成长课程）
- 如果使用其他格式/文件名，请同步更新 `index.html` 中相应的 `<img>` 路径。

1. **导航栏** - 固定顶部导航，带滚动效果

## 自定义修改

### 修改配色
在 `style.css` 的 `:root` 部分修改 CSS 变量:
```css
:root {
    --primary-color: #00d4ff;
    --secondary-color: #7c3aed;
    --accent-color: #ff6b6b;
    /* ... */
}
```

### 修改内容
直接编辑 `index.html` 中对应的文本内容

### 添加更多功能
在 `script.js` 中添加自定义JavaScript代码
